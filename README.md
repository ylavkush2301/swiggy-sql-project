# Swiggy SQL Sales Analysis Project

## Project Overview
This is a beginner-level SQL data analysis project based on Swiggy sales data.
The purpose of this project is to practice SQL queries and understand sales analysis.

## Tools Used
- SQL Server Management Studio (SSMS)
- SQL
- GitHub

## Dataset Description
The dataset contains sales-related information such as:
- Orders
- Customers
- Restaurants
- Order values

## Analysis Performed
- Created tables using CREATE TABLE
- Inserted records using INSERT INTO
- Retrieved data using SELECT queries
- Analyzed total sales and order details

## How to Run the Project
1. Open SQL Server Management Studio (SSMS)
2. Create a new database
3. Open the SQL file from this repository
4. Execute the queries step by step

## Learning Outcome
- Basic understanding of SQL
- Hands-on experience with SQL queries
- Learned how to upload and manage projects on GitHub
